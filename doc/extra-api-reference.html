    Matrix

    Matrix is a linear algebra library for manipulating dense matrices. Its design goals are to be reasonably efficient in terms of both memory usage and computations. Unfortunately there is a trade off between memory efficiency and computational efficiency. Where these requirements conflict Matrix will use the more computationally efficient algorithm
